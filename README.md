# TUNet CLI

version 0.0.1

by 思無邪SyiMyuZya

----

清華校園網命令行版用戶認證工具。

用於無圖形界面之環境。可顯示當前在線狀態、登入登出校園網。

需要python2(>=2.6)以執行。

1.  顯示在線狀態

    ```sh
    ./tunetctl
    ```

2.  登入

    ```sh
    ./tunetctl -l  # （提示輸入用戶名、密碼）
    
    # 使用文件保存的用戶信息（計劃中，尚未實作）：
    ./tunetctl -lf user.txt
    ```

3.  登出

    ```sh
    ./tunetctl -o
    ```

4.  顯示幫助

    ```sh
    ./tunetctl -h
    ```
